# Web Praktikum Pemrograman Web I

Project ini dibagi per pertemuan supaya lebih mudah dipelajari dan dikumpulkan.

## Struktur Folder

- `pertemuan-12/` — Variabel, output JavaScript, operator aritmatika/penugasan/pembanding.
- `pertemuan-13/` — If else, switch case, function, dan kalkulator HTML/CSS/JS.
- `pertemuan-14/` — Form data mahasiswa dengan fitur tambah, edit, update, hapus.
- `pertemuan-16/` — Simulasi CRUD mahasiswa, login, API, dan AJAX versi HTML/CSS/JS menggunakan localStorage.
- `pertemuan-17/` — Class JavaScript, inheritance, dan game kotak bergerak.

## Cara Menjalankan

1. Extract file ZIP.
2. Buka folder pertemuan yang ingin dicoba.
3. Klik dua kali file `index.html` atau jalankan dengan Live Server di VS Code.

## Catatan

Pertemuan 16 pada PDF aslinya menggunakan PHP dan MySQL. Di versi ini dibuat ulang menggunakan HTML, CSS, dan JavaScript murni agar bisa langsung dijalankan di browser tanpa XAMPP.
